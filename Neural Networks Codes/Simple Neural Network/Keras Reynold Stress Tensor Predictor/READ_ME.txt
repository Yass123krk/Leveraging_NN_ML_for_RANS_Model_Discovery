Reynolds Stress Tensor Predictor with Keras

Introduction:
This script utilizes Keras, a high-level neural networks API, to predict Reynolds stress tensor components based on wall distance and Reynolds number. It trains separate models for each stress tensor component and evaluates their performance using mean squared error (MSE)

Dependencies:

pandas
numpy
matplotlib
scikit-learn
Keras

This code was created in a SPYDER environment using ANACONDA
